Getting started
Three steps. That's it.

No language packs, no admin rights, no complex setup.

01 — Install

Run the installer

All 12 languages bundled with Indic fonts. Under 50 MB total.

02 — Toggle

Press F11 or F12

Switch between English and your chosen Indic script. Or press Ctrl+1-9 for a specific language.

03 — Type

Type naturally

Type namaste and get नमस्ते. Works in every Windows app — Word, Chrome, WhatsApp, Notepad.

Features
Built for real typing.
🖥️

System tray app

Lives in your taskbar. Auto-starts with Windows. No window needed.

⚡

Instant switching

F11/F12 toggle. Ctrl+1 through Ctrl+= for direct language pick. Toast notification confirms.

🔗

Conjuncts

kka → क्क, shri → श्री, jña → ज्ञ. Full conjunct and ligature support.

🧠

Smart schwa

Hindi-family: implicit schwa (no trailing halant). Sanskrit: explicit halant. Per-language behavior.

🌍

Works everywhere

Any Windows app — Word, Notepad, Chrome, WhatsApp Desktop, VS Code. If it takes text, it works.

📖

Open source

MIT licensed. Read, fork, modify. No telemetry, no ads, no accounts.
