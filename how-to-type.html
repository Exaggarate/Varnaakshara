How to Type

A complete reference for Varnaakshara's phonetic input system. Type English letters, get Indian script — see every mapping below.

Quick Start
Type English letters phonetically to produce Indic script output.
Basics
Type phonetically: namaste → नमस्ते
Use aa or A for long vowels: rAma → राम
Capitals = Retroflex / Aspirated
T=ट Th=ठ D=ड Dh=ढ N=ण
K=ख G=घ S=श Sh=ष
Long Vowels
Double the vowel: aa=आ ee=ई oo=ऊ
Or capitalize: A=आ I=ई U=ऊ
Conjuncts
Type consonants together: kka=क्क shri=श्री
Special: kSh=क्ष j~j=ज्ञ
